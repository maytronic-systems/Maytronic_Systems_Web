Android App

IRIS

A privacy-oriented dashboard that visualizes how apps use permissions and data — so you can make informed choices.

Get IRIS Features

Note: IRIS focuses on analysis and visualization. It does not claim to block, remove, or interfere with other apps.

Core modules

IRIS is structured into modules so you can focus on what matters: permissions, traffic, and usage patterns.

Permission Monitor

Organized permission groups and clear summaries per app. Designed for readability and quick comparisons.

App Statistics

Network traffic and usage views to help you understand where data volume and time are spent.

Action Center (Pro)

Configurable alerts for activity changes, permission changes, and background activity — as user-defined monitoring rules.

What you get

App Activity Monitor

  • Per-app overview: permissions + activity signals
  • Optional grouping by categories
  • Designed for neutral interpretation (no “threat labeling”)

Network Traffic Monitor

  • Visual breakdown (e.g., top apps + “Others”)
  • Helps you quickly spot high-volume apps
  • Works as an analytics dashboard

Usage Monitor

  • Foreground / background usage views
  • Time-based summaries
  • Designed for clarity and speed

Action Center (Pro)

  • Network activity alerts (rules)
  • Permission change alerts (rules)
